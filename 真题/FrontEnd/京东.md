### 1. 冒泡排序 <br>
重复的比较相邻的两个元素 <br>
比较的外循环次数为n-1 <br>
```javascript
function bubbleSort(array){
    var l = array.length;
    var i, j, temp;
    for(i = l - 1; i > 0; i--){
        for(j = 0; j < i; j++){
            if(array[j] > array[j+1]){
                temp = array[j];
                array[j] = array[j+1];
                array[j+1] = temp;
            }
        }
    }
    return array;
}
```
