### 同向两根指针 <br>
- 【move zeros】<br>
把数组中的0移到末尾 <br>
left指向第一个0，如果right不为0，交换left与right位置上的数，left++
```java
    public void moveZeroes(int[] nums) {
        int left = 0, right = 0;
        while(right < nums.length) {
            if(nums[right] != 0) {
                // swap
                int temp = nums[right];
                nums[right] = nums[left];
                nums[left] = temp;
                left++;
            }
            right++;
        }
    }
```
